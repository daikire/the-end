# Research フォルダ — 研究開発

> PM方法論、業界知識、技術研究

## このフォルダの目的

PMO業務の専門性を深掘りするための研究・学習ノートです。PM方法論（アジャイル、リーン、等）、証券代行業務の業界動向、関連技術の調査、ベストプラクティス事例研究などを記録します。理論と実践をつなぎ、業務改善のための知見源となります。

## サブフォルダ構成

| フォルダ | 用途 |
|---------|------|
| methodologies/ | PM方法論、フレームワーク、プロセス設計理論 |
| industry/ | 証券代行業務の業界知識、監規動向、事例研究 |
| tech/ | 技術調査、ツール評価、新技術の影響分析 |
| references/ | 参考文献、リンク集、引用元管理 |

## ファイル命名規則

- `research_[分野]_[テーマ]_YYYYMMDD.md` — 研究ノート
- `industry_[トピック]_[出典].md` — 業界情報
- `tech_evaluation_[ツール名]_v[バージョン].md` — 技術評価

## 運用ルール

- 学習内容は週1回以上、新たな理解を得たら随時記録する
- 業界情報は出典と取得日を必ず明記する
- 技術評価は実装前後で比較検証を行う
- 参考文献はURLまたは図書情報を正確に記録する

## 知見の蓄積方法

月ごとに研究ノートを分類し、他フォルダで実装可能な知見をハイライトします。業界報告書や論文から抽出した重要概念を、現在の業務課題とマッピングし、改善案につなげます。
