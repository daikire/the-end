# CEO フォルダ — プロジェクト統括

> キャリア全体の方向性と大局的な判断を司る

## このフォルダの目的

このフォルダは個人プロジェクト全体の羅針盤です。証券代行業務のPMOとしてのキャリアビジョン、5年・10年スパンの目標、重要な経営判断、予算管理、重要な契約やステークホルダー関係を記録・管理します。戦略的な意思決定はここから生まれます。

## サブフォルダ構成

| フォルダ | 用途 |
|---------|------|
| vision/ | キャリアビジョン、ミッション、パーパス |
| decisions/ | 重要な経営判断、意思決定記録 |
| budget/ | 予算管理、投資計画、リソース配分 |

## ファイル命名規則

- `vision_2026-2031.md` — 中期ビジョン（年号または期間を含める）
- `decision_YYYYMMDD_[内容].md` — 意思決定記録（日付と判断内容）
- `budget_FY2026.md` — 予算計画（会計年度を含める）
- `opm_q2_2026.md` — 四半期OKR（OPM = Personal OKR）

## 運用ルール

- 意思決定は必ず判断時点で記録する。後付けはしない
- 予算案件は実行前にこのフォルダで承認フロー完了させる
- ビジョンは半期に1回、実現度と修正必要性を振り返る
- CEO判断で他フォルダの優先順位を変更できる権限を持つ

## 知見の蓄積方法

意思決定の背景、判断基準、結果を「決定→実行→振り返り」のサイクルで記録します。失敗した判断、予想外の結果も含めて記述し、次の判断精度向上に活用します。
