# 09_hr — 人材管理・チームマネジメント

将来のメンバーマネジメントに備え、マネジメント哲学・手法・記録を蓄積するフォルダ。

**現在のフェーズ**: 準備期（後輩・メンバーが来る前の学習・設計期間）

---

## フォルダ構成

| フォルダ | 用途 |
|---------|------|
| members/ | メンバー個別の記録・成長管理（後輩ができたら作成） |
| oneonone/ | 1on1ミーティングの記録・テンプレート |
| frameworks/ | マネジメント手法・評価フレームワーク |
| growth/ | メンバー育成計画・スキルマップ |

---

## マネジメント哲学（Daikiの考え方）

> 「人生に目的を持って生きる」を支援するマネジメントを目指す

- メンバーの強みを見つけて伸ばす
- 失敗を責めず、なぜなぜで一緒に考える
- キャリアの目標を共有し、仕事の意味を一緒に作る
- 自分が経営者を目指す立場だからこそ、育てる側の視点を早期から養う
