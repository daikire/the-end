# Marketing フォルダ — 自己ブランディング

> 個人ブランディング、プレゼンテーション資料、キャリア開発

## このフォルダの目的

「マーケティング」は通常の営業推進ではなく、PMOとしての個人ブランドを構築する領域です。業界での発言権、専門性の可視化、キャリアアップに向けた自己啓発資料、登壇資料、ポートフォリオなどを管理します。証券代行業務での専門性を外部に発信し、キャリアの価値を高めます。

## サブフォルダ構成

| フォルダ | 用途 |
|---------|------|
| brand/ | 自己紹介資料、プロフィール、ポートフォリオ |
| presentations/ | 登壇スライド、社内プレゼンテーション資料 |
| self-development/ | 学習ロードマップ、資格取得計画、技能習得記録 |

## ファイル命名規則

- `brand_profile_v[バージョン].md` — プロフィール文（定期更新）
- `presentation_[イベント名]_YYYYMMDD.pptx` — プレゼン資料
- `development_[スキル名]_roadmap.md` — 開発ロードマップ

## 運用ルール

- プロフィールは年2回以上アップデートする
- 登壇履歴や実績は実施後1週間以内に記録する
- 開発ロードマップは四半期ごとに進捗を振り返る
- 対外発言資料は事前に信頼性を確認する

## 知見の蓄積方法

登壇後のフィードバック、反応の良かったコンテンツを分析します。複数のプレゼンから共通する質問や関心分野を抽出し、次の学習テーマや発信テーマに反映させます。
