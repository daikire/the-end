# 08_cfo — 資産管理・投資

個人CFO（最高財務責任者）として、資産形成・投資の記録・分析・学習を行うフォルダ。

**重要な前提**: このフォルダで行うのは「情報の整理・記録・学習」であり、投資判断は自己責任で行う。
Claudeは情報整理と分析補助はできるが、「この銘柄を買え」等の投資推奨は行わない。

---

## フォルダ構成

| フォルダ | 用途 |
|---------|------|
| strategy/ | 投資方針・長期戦略・アセットアロケーション |
| portfolio/ | 保有資産・ポートフォリオの記録・推移 |
| analysis/ | 銘柄分析・市場分析のメモ |
| learning/ | 投資の学習ノート・書籍メモ・用語集 |
| risk/ | リスク管理・損失記録・反省 |

---

## 運用ルール

1. **方針ファイルを最初に作る** → `strategy/investment-policy.md`
2. **月次でポートフォリオを更新** → `portfolio/YYYY-MM.md`
3. **学習した内容はその都度記録** → `learning/YYYY-MM-DD-topic.md`
4. **損失・失敗も正直に記録する** → `risk/` に残す（成長の源泉）
5. **感情で動かない** → 方針ファイルを読み返してから判断する
